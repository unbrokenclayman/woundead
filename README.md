# Vue + Pug + Stylus
Vue boilerplate

Post about this stack can be found [here](https://medium.com/@wesselvanderpal/vue-components-with-pug-and-stylus-556508b74c96)

*Component structure:*

```
├── config.json
├── index.vue
├── script.js
├── style.styl
├── template.pug
└── test.spec.js
```

### Install

`yarn`

### Dev

`yarn dev`

### Test

`yarn test`
